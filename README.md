# News Article Q&A using RAG

A simple Retrieval-Augmented Generation (RAG) project built using LangChain, ChromaDB, and Groq.

## What it does

* Loads news articles from text files
* Splits articles into chunks
* Creates embeddings for semantic search
* Stores embeddings in ChromaDB
* Retrieves relevant content based on user queries
* Uses an LLM to generate answers from retrieved context

## Tech Stack

* Python
* LangChain
* ChromaDB
* Groq
* Jupyter Notebook

## Example Query

"How much money did Microsoft raise?"

The system retrieves relevant article chunks and generates an answer based on the retrieved information.

## Key Learnings

* Retrieval-Augmented Generation (RAG)
* Vector Databases
* Embeddings
* Semantic Search
* LangChain Workflows
* LLM Integration
